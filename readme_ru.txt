разрешения


TransBoxEnhanced — это простой инструмент, способный запускать и создавать файлы .reg, которые изменяют прозрачность/выделение поля.
на окнах.


Ключ реестра, который изменяет эта программа: HKEY_CURRENT_USER\Control Panel\Colors --- «Hilight» и «HotTrackingColor».



Гид:

- Откройте main.py или запустите TransBoxEnhanced.exe (создано с помощью pyinstaller)

- Вручную введите значения RGB в соответствующие поля ИЛИ выберите собственный цвет с помощью палитры цветов внизу.
правый угол

- Создайте И ИЛИ. Запустите .reg для этого конкретного цвета.
    - «Создать И запустить .reg (Администратор)» = автоматически создает и запускает .reg ***Требуется либо 1) запуск от имени администратора, либо
2) запускать в папке, которая не требует администратора ***
    - «Создать файл .reg» = просто создайте файл .reg для запуска вручную или для дальнейшего использования.


- «Открыть палитру цветов» = открывает пользовательский селектор цвета и применяет его к входам RGB при нажатии «ОК».


- «Просмотр цвета (если введен вручную) = позволяет просматривать цвет необработанных значений RGB, если они введены вручную.

- «Значения по умолчанию» = вводит цвет окна по умолчанию (0 120 215) на входы RGB. (Или запустите «DefaultValues.reg в другой папке)
